## 📋 Sobre

Repositório dedicado aos estudos da linguagem **Java**. Contém exercícios práticos abordando desde conceitos fundamentais até tópicos avançados da linguagem.

### 🧠 Tópicos Abordados

- ✅ **Orientação a Objetos** — Classes, herança, polimorfismo, interfaces
- ✅ **Collections Framework** — List, Set, Map e suas implementações
- ✅ **Streams API** — Programação funcional com Java
- ✅ **Boas práticas** — Padrões de código, tratamento de exceções

---

## 🚀 Como Usar

```bash
git clone https://github.com/ismaeldouglasdev/Java-studies.git
```

Abra a pasta `My-java-studies` na sua IDE de preferência (IntelliJ, Eclipse, VS Code).

### Pré-requisitos

- Java 17+ (JDK)
- Uma IDE Java (recomendado: IntelliJ IDEA)
